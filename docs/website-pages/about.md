# About Page Specification - Elysian Flame

## 1. Page Purpose
Tell the Elysian Flame brand story, build emotional connection, communicate values (handmade, natural, sustainable), and differentiate from mass-produced candles.

## 2. Business Goal
- Build brand trust and authenticity
- Increase "Add to Cart" rate from visitors who view About page (they're more qualified)
- Support SEO with brand story keywords
- Provide shareable content for PR/media

## 3. Target User Intent
- Who makes these candles?
- What makes them special/different?
- What are the brand values?
- Can I trust this brand?
- What's the origin story?

## 4. Theme & Visual Rules
- `#6CA651` - Section headings, key highlights
- `#BBCB2E` - Timeline markers, value proposition icons
- `#839705` - Pull quotes, subheadings
- Botanical photography, behind-the-scenes images, founder photo

## 5. Layout Structure
1. Hero: Hero statement + founder image
2. Origin Story: Text + imagery, 2-column layout
3. Our Values: 4 icon cards (Natural, Handmade, Sustainable, Community)
4. Process: How candles are made (illustrated steps or video)
5. Meet the Maker: Founder bio + photo
6. Sustainability Commitment: Eco-friendly practices
7. CTA: "Experience the Difference" → Shop button

## 6. Component List
- **AboutHero** - Opening statement
- **StorySection** - Rich text + images
- **ValueCard** - Icon + heading + description
- **ProcessTimeline** - Step-by-step candle creation
- **FounderBio** - Personal story + photo
- **SustainabilityStats** - Key metrics (e.g., "10,000 lbs soy wax used")
- **ShopCTA** - Link to collections

## 7-21. [Condensed for length - follows same detailed structure as other pages]

**SEO Focus:** Brand name keywords, "handmade soy candles", "sustainable candles", "small batch candles", founder name

**Analytics:** `About Page Viewed`, `Founder Bio Read`, `Shop CTA Clicked from About`
