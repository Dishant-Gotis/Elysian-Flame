# Elysian Flame - Handmade Soy Candles Website

A beautiful showcase website for Elysian Flame, featuring hand-poured natural soy candles with a green botanical theme.

## Tech Stack

- **Next.js 14** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **React** - UI library

## Features

- 🕯️ Full candle collection gallery
- 📱 Fully responsive design
- 🎨 Green botanical design theme
- 📧 Contact form for orders
- ⚡ Fast, optimized performance
- ♿ Accessible (WCAG 2.1 AA)

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm start
```

## Pages

- **Home** (`/`) - Hero, featured candles, value propositions
- **Gallery** (`/gallery`) - Full product collection
- **About** (`/about`) - Brand story and values
- **Contact** (`/contact`) - Order inquiry form

## Deployment

Deploy easily to Vercel:

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/yourusername/elysian-flame)

Or manually:

```bash
npm run build
```

Then deploy the `.next` folder to your hosting provider.
